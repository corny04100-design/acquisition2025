# GitHub Pages 배포 (docs 폴더)

1. 이 `docs` 폴더 전체를 저장소 루트에 넣습니다.
2. GitHub → Settings → Pages → **Source: `Deploy from a branch`**, **Branch: `main` / Folder: `/docs`** 로 설정합니다.
3. 약간의 배포 시간이 지난 후, Pages 주소로 접속하면 계산기를 볼 수 있습니다.

파일 목록:
- `index.html` (메인 페이지)
- `manifest.webmanifest` (PWA 메타)
- `icon-192-v5.png`, `icon-512-v5.png` (아이콘)
